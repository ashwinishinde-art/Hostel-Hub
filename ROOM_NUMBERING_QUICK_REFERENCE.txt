┌─ ROOM NUMBER PATTERN ─────────────────────────────────────────────────────┐
│                                                                             │
│  Format: {FLOOR}{POSITION:02d}                                            │
│                                                                             │
│  Examples:                                                                  │
│  • Floor 2, Position 1  → Room 201                                        │
│  • Floor 2, Position 3  → Room 203                                        │
│  • Floor 3, Position 5  → Room 305                                        │
│  • Floor 10, Position 2 → Room 1002                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ QUICK FACTS ─────────────────────────────────────────────────────────────┐
│                                                                             │
│  ✓ Maximum 5 rooms per floor                                              │
│  ✓ Room numbers auto-generated (no manual entry)                          │
│  ✓ Position must be 1-5 (valid range)                                     │
│  ✓ Room numbers must be unique across hostel                              │
│  ✓ Valid for any floor number (1-digit or multi-digit)                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ ADDING A ROOM ───────────────────────────────────────────────────────────┐
│                                                                             │
│  1. Click "Add New Room" button in admin dashboard                         │
│  2. Select Floor (e.g., 2)                                                │
│  3. Select Position (dropdown: 1, 2, 3, 4, 5)                             │
│  4. System shows generated room number (e.g., Room 201)                    │
│  5. Fill in details: Type, Capacity, Rent, Amenities                      │
│  6. Click "Add Room"                                                       │
│  7. Success: "Room 201 added successfully!"                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ EDITING A ROOM ──────────────────────────────────────────────────────────┐
│                                                                             │
│  1. Click "Edit" button on existing room (e.g., Room 203)                  │
│  2. Current room number shown (read-only): 203                             │
│  3. Can change Floor and/or Position                                       │
│  4. New room number preview shown (e.g., 305)                              │
│  5. Modify other details as needed                                         │
│  6. Click "Update Room"                                                    │
│  7. Success: "Room 305 updated successfully!"                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ VALIDATION RULES ────────────────────────────────────────────────────────┐
│                                                                             │
│  POSITION VALIDATION:                                                       │
│  • Must be between 1 and 5                                                 │
│  • Error: Position 0 ✗ Invalid                                             │
│  • Error: Position 6 ✗ Invalid                                             │
│  • Error: Position -1 ✗ Invalid                                            │
│                                                                             │
│  MAX ROOMS VALIDATION:                                                      │
│  • Each floor can have maximum 5 rooms                                     │
│  • Error: "Maximum 5 rooms allowed per floor"                              │
│  • Enforced at application AND database level                              │
│                                                                             │
│  UNIQUENESS VALIDATION:                                                     │
│  • Room numbers must be unique                                             │
│  • Error: "Room 201 already exists"                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
