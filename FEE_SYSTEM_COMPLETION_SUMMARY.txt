═════════════════════════════════════════════════════════════════════════════════
                          HOW TO USE THE SYSTEM
═════════════════════════════════════════════════════════════════════════════════

FOR ADMINS:
──────────
1. Login with admin credentials
2. Navigate to Admin Dashboard → Fee Management (or /admin/fees)
3. Add fees using the form at the top
4. Choose to apply to all students or select specific ones
5. Click "Add Fees" button
6. View all fees in the table below
7. To record a payment, click "Record" button next to any fee

FOR STUDENTS:
─────────────
1. Login with student credentials
2. Navigate to Student Dashboard → Fee Management (or /student/fees)
3. View your assigned fees in the table
4. To pay, click the "Pay" button on any unpaid fee
5. Enter payment amount (max = pending amount)
6. Select payment method
7. Click "Confirm Payment"
8. View payment history below the fees table
