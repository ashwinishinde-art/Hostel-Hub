## ❓ Common Questions & Answers

### Q1: What if I can't find the "Browse Files" button?
```
Make sure you:
1. ✓ Logged in as admin
2. ✓ On the gallery page (/admin/gallery)
3. ✓ In the "Add New Photo" tab
4. ✓ Below "Image Path" input field
5. ✓ Right side of the input
```

### Q2: What file types work?
```
✅ YES:
   • JPG / JPEG
   • PNG
   • GIF
   • WebP

❌ NO:
   • PDF
   • Word docs
   • Videos
   • Audio files
```

### Q3: What if the file is too large?
```
Error message:
❌ "File size should be less than 5MB"

Solution:
1. Reduce image size
2. Use online compressor
3. Edit with photo app
4. Try a different image
```

### Q4: Can I drag & drop instead?
```
YES! Both methods work:

METHOD 1:
Click "Browse Files" → Select → Done

METHOD 2:
Open file manager → Drag image to zone → Done

METHOD 3:
Click on upload zone → Select → Done
```

## ⚠️ Troubleshooting

### Issue: File Manager Doesn't Open

```
Try these:
1. ✓ Refresh the page
2. ✓ Close and reopen browser
3. ✓ Try different browser
4. ✓ Check browser permissions
5. ✓ Clear browser cache
```

### Issue: Upload Fails

```
Check:
1. ✓ File is less than 5MB
2. ✓ File is an image (JPG, PNG, etc)
3. ✓ Title is filled in
4. ✓ Internet connection active
5. ✓ Admin account logged in
```

### Issue: Preview Not Showing

```
Possible causes:
1. Large file (may take time)
2. Browser issue (try refreshing)
3. Network issue (try again)
4. Corrupted file (try another)
```
