Available Variables (auto-adapt to theme):

Text Colors:
  --text-primary      : Main text color (black in light, white in dark)
  --text-secondary    : Secondary text (gray in light, light gray in dark)
  --text-tertiary     : Tertiary text (lighter gray in light, lighter in dark)

Background Colors:
  --bg-primary        : Main background (white in light, dark in dark)
  --bg-secondary      : Secondary background (light gray in light, dark in dark)
  --bg-tertiary       : Tertiary background (lighter in light, darker in dark)

Other Colors:
  --border-color      : Borders (auto-adjusted per theme)
  --primary           : Primary accent color
  --success           : Green for success
  --warning           : Orange for warnings
  --danger            : Red for errors
  --info              : Blue for info

❌ BAD - Text is NOT visible:
   - Black text on dark background
   - White text on white background
   - Gray text that blends with background

================================================================================
                         TROUBLESHOOTING
================================================================================

ISSUE: Colors don't change when switching themes
SOLUTION: Clear browser cache
  Chrome/Edge: Ctrl+Shift+Delete
  Firefox: Ctrl+Shift+Delete
  Safari: Cmd+Shift+Delete

ISSUE: Some text still invisible
SOLUTION: Check that specific element for hardcoded colors
  - Search in template files for "color: #" or "color: white"
  - Replace with CSS variable name

ISSUE: Theme doesn't match system preference
SOLUTION: Check browser DevTools
  - F12 → Console for JavaScript errors
  - Check if @media (prefers-color-scheme) is working
